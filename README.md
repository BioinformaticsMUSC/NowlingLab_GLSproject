Cytokine-driven glycosphingolipid metabolism modulates ER stress response in primary human renal mesangial cells
==========================

This repository contains analysis code for the bulk RNA-seq project carried out by researchers at the [Nowling Lab, MUSC](https://medicine.musc.edu/departments/dom/divisions/rheumatology-and-immunology/research/labs-and-centers/nowling-lab) and [Berto Lab, MUSC](https://bertolab.org/)
